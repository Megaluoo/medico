# medico

Módulo inicial para gestión de recetas y certificados médicos. Incluye:

- **Interfaz Tailwind** para redactar recetas, constancias, reposos e informes.
- **CertificatesController** con métodos `index`, `create`, `list`, `generate` y `download`.
- **Plantillas HTML** base para los PDFs sin generar archivos binarios.
- **Rutas** preparadas en `routes/web.php` para `/certificates`.
